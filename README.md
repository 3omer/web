# Deployment Guides

- Install NodeJs .
- Open Terminal
- Go to your file project
- Run in terminal: `npm install -g @angular/cli`
- Then: `npm install`
- And: ng serve
- Navigate to: http://localhost:4200/dashboard and   http://localhost:4200/dashboard/balance-inquiry
